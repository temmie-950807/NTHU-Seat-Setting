# 電腦教室座位・密碼產生器

一站式的純前端網頁工具，協助助教在上機考／實習前完成：**座位安排 → 密碼產生 → 座位表列印 → 密碼紙列印**。所有運算都在瀏覽器端，無需後端。

## 功能

| 分頁 | 說明 |
|------|------|
| **① 資料輸入** | 貼上或上傳學生名單，勾選啟用的教室（326 / 328），一鍵產生密碼並隨機排座 |
| **② 座位安排** | 隨機、前排優先的座位圖；可**拖拉**互換或移動、重新洗牌、停用特定座位，並直接列印 |
| **③ 密碼管理** | 姓名／帳號／密碼表格，可單筆或全部重產，匯出 `passwords.csv` |
| **④ 密碼紙** | 鏡射教室座位的網格，沿虛線裁切、走道以粗線分隔，方便影印後分發（建議橫向列印） |

## 快速開始

```bash
npm install      # 安裝相依套件
npm run dev      # 啟動開發預覽（瀏覽器開啟顯示的 localhost 位址）
npm run build    # 產生 dist/ 靜態檔，可直接部署或本機開啟
```

## 使用流程

1. 於「資料輸入」貼上名單，每行一位，格式 `姓名,帳號`（以逗號或 Tab 分隔）。
   - 帳號留空會自動補 `team01`、`team02`…；只填姓名亦可。
2. 勾選這次要使用的教室，按「**產生密碼並排座**」。
3. 到「座位安排」拖拉微調、停用不用的座位，或按「重新洗牌」重排。
4. 於「密碼管理」匯出 `passwords.csv`；於「密碼紙」列印裁切。

## 教室配置

- **326 教室**：每排 7 座，分左 3 ｜走道｜右 4；共 8 排（節奏 1 / 2 / 2 / 2 / 1），最多 56 座。
- **328 教室**：每排 7 座，分 2 ｜走道｜2 ｜走道｜3；共 7 排（排間無間距），最多 49 座。
- 兩間皆面向頂端的投影幕，座位由前排（近投影幕）開始安排。

版面定義集中在 `src/data/roomLayouts.ts`，可自行調整排數與分區。

## 密碼規則

- 每人一組 10 碼英數。
- 排除易混淆字元（`0 O o 1 l I i`），並確保全體互異。

## 儲存與還原

- 資料會自動存於瀏覽器 `localStorage`，重整不遺失。
- 右上角「**匯出設定 / 匯入設定**」可備份或還原**完整座位安排**（含拖拉微調、停用座位）。
- 注意：`passwords.csv` 僅含 `帳號,姓名,密碼`，**不含**座位資訊，無法用來還原座位。

## 技術

Vite + React + TypeScript，拖拉功能使用 [@dnd-kit/core](https://dndkit.com/)。
